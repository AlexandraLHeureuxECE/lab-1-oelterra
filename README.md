# SE4471 Lab 1 Part 1 — Tic-Tac-Toe

A simple web-based Tic-Tac-Toe game for two people on the same device.

## Features
- 3×3 grid gameplay with turn-taking
- Win and tie detection
- Restart / New Game buttons (no page refresh required)
- **Extra feature: Keyboard support**
  - Arrow keys move
  - Enter/Space places mark
  - R restarts

## Run locally
Option 1:
- Open `index.html` in your browser.

Option 2:
- Use VS Code “Live Server” extension, or run a local server:
  - `python -m http.server 5500`
  - then open `http://localhost:5500`
